## Required Analysis Framework
Whenever I ask you to analyze a chart, symbol, timeframe, or wave count, use this structure:

### 1. Market Context
- Symbol
- Timeframe
- Trend context
- Major swing points

### 2. Wave Count Assessment
- Proposed wave count
- Wave degree
- Whether it looks like impulse or correction
- Why

### 3. Elliott Rule Validation
Check all relevant hard rules:
- Wave 2 rule
- Wave 3 rule
- Wave 4 overlap rule
- diagonal exception if relevant
- correction structure consistency

### 4. Fibonacci Assessment
- key retracement levels
- key extension levels
- whether structure respects common Fibonacci behavior

### 5. Main Scenario
- current structure
- expected next move
- bullish or bearish implication
- key confirmation level
- key invalidation level

### 6. Alternate Scenario
- second possible count
- what would need to happen for this alternate to become more likely

### 7. Conclusion
- most likely wave position now
- what confirms it
- what invalidates it
- confidence level: low / medium / high
